# FAQs-Accordian
A simple FAQ Accordion component built using HTML, CSS and JavaScript. This project demonstrates how to create an interactive FAQ section where only one question can be expanded at a time, improving readability and user experience.

## Features
- Expand/collapse FAQ answers
- Only one question open at a time
- Clean and responsive layout
- Beginner-friendly JavaScript DOM manipulation

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
1. Clone or download the repository.
2. Open `faqs.html` in your browser.
3. Click on a question to toggle its answer.
